# AdvancedAdminTools Plugin Documentation

## Commands

| Command | Description | Permission |
|---------|-------------|------------|
| `/ban` | Ban a player from the server. | `aat.ban` |
| `/mute` | Mute a player in the chat. | `aat.mute` |
| `/warn` | Warn a player. | `aat.warn` |
| `/unban` | Unban a player from the server. | `aat.unban` |
| `/unmute` | Unmute a player in the chat. | `aat.unmute` |
| `/tools` | Open the admin tools menu. | `aat.tools` |

## Usage

- `/tools` — Open the admin tools GUI.
- `/mute <player> <time> <reason>` — Mute a player for a specific duration with a reason.
- `/unmute <player>` — Unmute a player.
- `/warn <player> <reason>` — Warn a player with a reason.
- `/ban <player> <time> <reason>` — Ban a player for a specific duration with a reason.
- `/unban <player>` — Unban a player.
